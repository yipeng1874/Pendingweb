# todo-h5

独立于现有 PC 项目的 H5 我的待办前端。

## 当前已实现
- 账号密码登录
- 飞书登录入口与回调处理
- 登录后身份选择
- 我的待办列表
- 待办详情基础展示

## 当前原则
- 不改现有 `frontend` 项目页面文件
- 复用现有后端 API
- 先跑通 `登录 -> 身份 -> 我的待办` 主链路

## 启动
```bash
cd todo-h5
npm install
npm run dev
```

默认开发端口：
- `http://localhost:4173`

代理后端：
- `/api -> http://127.0.0.1:4000`
